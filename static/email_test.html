Troubleshooting Guide

Common Issues and Solutions:

1. App Password Format: Should be exactly 16 characters with no spaces or dashes
2. 2-Factor Authentication: Must be enabled on your Gmail account
3. Account Mismatch: App Password must be generated for qrlegends22@gmail.com
4. Spaces/Dashes: Remove all formatting when copying the App Password
5. Old Password: Generate a new App Password if the current one is old

Generate New App Password:
1. Go to myaccount.google.com/apppasswords
2. Select "Mail" and generate new password
3. Copy the 16-character password (no spaces/dashes)
4. Update GMAIL_APP_PASSWORD in Replit Secrets
5. Restart your application
