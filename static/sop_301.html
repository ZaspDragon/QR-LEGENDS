📊 SOP-301: Cycle Counting Procedures

Standard Operating Procedure for Cycle Count Operations

Purpose

To maintain accurate inventory records through systematic cycle counting that minimizes business disruption while ensuring inventory accuracy.

Cycle Count Schedule

ABC Classification System

  • A Items (High Value): Monthly cycle count
  • B Items (Medium Value): Quarterly cycle count
  • C Items (Low Value): Semi-annual cycle count
  • Fast Moving Items: Monthly regardless of value

Count Frequency Triggers

  • Scheduled cycle count dates
  • Negative inventory balances
  • Large quantity adjustments
  • Customer complaints about availability
  • Physical damage observations

Pre-Count Preparation

System Preparation

  • Generate cycle count worksheets from QR Legends system
  • Freeze inventory transactions for count locations
  • Print location tags and count sheets
  • Verify last transaction dates

Physical Preparation

  • Ensure count areas are clean and organized
  • Complete any pending put-away activities
  • Remove empty containers and debris
  • Verify location signage is visible and accurate

Count Execution

Initial Count Process

  • Scan location QR code using mobile device
  • Count all items in designated location
  • Record quantities by part number
  • Note any damaged or questionable items
  • Photograph discrepancies or damage

Count Verification

  • Second person independently verifies large discrepancies
  • Recount items with variances over threshold
  • Investigate obvious errors or unusual findings
  • Confirm part numbers and descriptions

Documentation Requirements

  • Sign and date all count sheets
  • Enter counts into QR Legends system immediately
  • Document reasons for any variances
  • Photograph problem areas for analysis

Variance Analysis

Threshold Guidelines

  • Zero tolerance for high-value items (>$1000)
  • ±2% variance acceptable for medium-value items
  • ±5% variance acceptable for low-value items
  • Any variance in controlled or regulated items

Root Cause Investigation

  • Review recent transaction history
  • Check for timing differences
  • Verify unit of measure conversions
  • Investigate potential shrinkage causes
  • Review handling and storage procedures

Adjustment Process

Approval Requirements

  • Supervisor approval for adjustments over $500
  • Manager approval for adjustments over $2000
  • Director approval for adjustments over $10000
  • Document business justification for all adjustments

System Processing

  • Enter approved adjustments in QR Legends system
  • Generate adjustment reports for accounting
  • Update ABC classifications if needed
  • Schedule follow-up counts for problematic items

Performance Metrics

  • 95% inventory accuracy target
  • Complete cycle counts within 24 hours
  • Zero production delays due to inventory errors
  • Monthly shrinkage under 0.5% of total inventory value

Equipment and Tools

  • QR Legends mobile counting device
  • Portable scale for weight verification
  • Measuring tools for dimensional items
  • Calculator for unit conversions
  • Camera for documentation
