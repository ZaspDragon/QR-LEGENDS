📄 SOP-203: Documentation Requirements

Standard Operating Procedure for Shipping Documentation

Purpose

To ensure all shipments include complete and accurate documentation required for delivery, customs clearance, and regulatory compliance.

Standard Documentation

Packing Slip/Invoice

  • Complete itemized list of shipped products
  • Quantities, part numbers, and descriptions
  • Unit prices and extended amounts
  • Order number and purchase order reference
  • Ship-to and bill-to addresses

Shipping Label

  • Complete delivery address with ZIP+4
  • Return address information
  • Tracking number and service type
  • Package weight and dimensions
  • Special handling labels as required

Certificate of Conformance (When Required)

  • Product specifications and test results
  • Manufacturing date and lot numbers
  • Quality assurance signatures
  • Compliance statements for applicable standards

International Shipment Documentation

Commercial Invoice

  • Detailed description of goods in English
  • Country of origin for each item
  • Harmonized tariff codes (HS codes)
  • Value declarations for customs
  • Terms of sale (Incoterms)

Export Documentation

  • Electronic Export Information (EEI) when required
  • Export license if controlled goods
  • End-user certificates as needed
  • NAFTA/USMCA certificates of origin

Restricted/Regulated Items

  • FDA documentation for food/medical devices
  • FCC certificates for electronic equipment
  • Safety data sheets (SDS) for chemicals
  • Phytosanitary certificates for plant materials

Special Documentation Requirements

Hazardous Materials

  • Shipper's Declaration for Dangerous Goods
  • Safety Data Sheets (SDS)
  • UN specification markings on packages
  • Emergency response contact information
  • Proper shipping names and classifications

High-Value Shipments

  • Detailed inventory with serial numbers
  • Insurance documentation and receipts
  • Chain of custody forms
  • Security sealing certificates

Temperature-Controlled Shipments

  • Temperature monitoring certificates
  • Cold chain compliance documentation
  • Handling instruction labels
  • Time-sensitive delivery requirements

Documentation Process

Pre-Ship Verification

  • Review all required documents using QR Legends checklist
  • Verify customer-specific documentation requirements
  • Check for regulatory compliance needs
  • Confirm all signatures and approvals are obtained

Document Preparation

  • Print all documents using approved templates
  • Ensure clear, legible print quality
  • Place documents in protective sleeves
  • Attach securely to shipment exterior

Record Keeping

  • Scan all documents into QR Legends system
  • Maintain digital copies for audit trail
  • Store physical copies per retention policy
  • Link documents to tracking numbers

Quality Control

  • 100% documentation completeness verification
  • Zero shipments delayed due to missing documents
  • Accuracy verification for all customer data
  • Compliance with all regulatory requirements
