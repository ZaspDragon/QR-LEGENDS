💡 How It Works

  1. Generate: Create a static QR code containing only a permanent UID
  2. Store: Item details are stored server-side linked to the UID
  3. Scan: When scanned, the UID fetches current data from the database
  4. Update: Change item details in the system without touching the QR code
  5. Track: All movements and changes are logged server-side using the UID
