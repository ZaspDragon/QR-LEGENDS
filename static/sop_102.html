📦 SOP-102: Batch Picking Guidelines

Standard Operating Procedure for Batch Picking Operations

Purpose

To optimize picking efficiency by combining multiple orders into batch picks while maintaining accuracy and order integrity.

Batch Creation Criteria

Order Grouping Rules

  • Similar delivery routes or zones
  • Compatible shipping methods
  • Similar priority levels
  • Maximum 8 orders per batch
  • No conflicting special requirements

Item Compatibility

  • No hazmat with non-hazmat items
  • Temperature requirements must match
  • Fragile items grouped separately
  • High-value items in secure batches

Batch Picking Process

Pre-Pick Setup

  • Scan batch ID using QR Legends app
  • Set up sorting containers for each order
  • Label containers with order numbers
  • Review batch summary for special instructions

Pick Execution

  • Follow optimized pick path
  • Pick total quantity for all orders at each location
  • Scan item and quantity using QR scanner
  • Place items in master container for sorting

Sorting and Allocation

  • Sort picked items by order number
  • Verify quantities match individual order requirements
  • Scan items into individual order containers
  • Double-check allocation accuracy

Quality Control

Verification Steps

  • Scan each sorted container to verify contents
  • Compare system counts with physical counts
  • Check for missed items or overages
  • Verify special handling requirements met

Error Resolution

  • Investigate any quantity discrepancies
  • Re-pick missing items immediately
  • Return excess items to proper locations
  • Document all corrections in system

Performance Standards

  • 30% improvement in pick efficiency over single-order picking
  • 99.5% allocation accuracy target
  • Maximum 2 hours total batch completion time
  • Zero order mix-ups between customers

Equipment Requirements

  • Multi-compartment pick cart or totes
  • QR Legends mobile device with batch picking module
  • Order labels and sorting containers
  • Handheld calculator for quantity verification
