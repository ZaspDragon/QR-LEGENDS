✅ SOP-103: Pick Accuracy Verification

Standard Operating Procedure for Verifying Pick Accuracy

Purpose

To ensure all picked orders are accurate and complete before moving to packing and shipping operations.

Verification Levels

Level 1: Self-Verification (Picker)

  • Scan each picked item before placing in container
  • Verify quantity matches pick list
  • Check item condition and quality
  • Confirm all items from pick list are collected

Level 2: Peer Verification (Checker)

  • Independent verification by different employee
  • Re-scan all items using QR Legends system
  • Physical count verification
  • Quality inspection for damage or defects

Level 3: Supervisor Verification (High-Value Orders)

  • Additional review for orders over $10,000
  • Serial number verification for tracked items
  • Documentation review for special requirements
  • Final authorization before packing

Verification Process

Setup and Preparation

  • Receive picked order container from picker
  • Scan order number to load verification screen
  • Set up clean, well-lit verification workspace
  • Ensure all verification tools are available

Item-by-Item Verification

  • Remove one item at a time from container
  • Scan item QR code using verification device
  • Confirm item description matches system
  • Count quantity and verify against pick list
  • Inspect for damage, expiration, or quality issues

Discrepancy Resolution

  • Stop verification process if discrepancy found
  • Document issue type and details in QR Legends system
  • Return to picker for correction if minor issue
  • Escalate to supervisor for complex discrepancies
  • Re-verify after corrections are made

Common Verification Checks

Quantity Verification

  • Count pieces/units match pick list exactly
  • Weigh items for weight-based products
  • Measure linear/area products as needed
  • Verify partial case quantities are correct

Quality Verification

  • No visible damage or defects
  • Packaging integrity maintained
  • Expiration dates within acceptable range
  • Product matches customer specifications

Documentation Verification

  • All special instructions noted and followed
  • Customer-specific labeling requirements met
  • Lot numbers and serial numbers recorded
  • Compliance certifications included if required

Performance Metrics

  • 99.8% verification pass rate target
  • Average 3 minutes per line item verification
  • Zero shipping errors due to verification failure
  • 100% compliance with verification procedures

Tools and Equipment

  • QR Legends verification scanner
  • Digital scale for weight verification
  • Measuring tools for dimensional products
  • Magnifying glass for small print inspection
  • Camera for documenting discrepancies
